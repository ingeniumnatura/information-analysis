PROBABILITY SPACES & RANDOM VARIABLES


A probability space is a measure space consisting of (Ω, 𝓕, P) where:


Ω, the sample space, is the set of all possible outcomes of a random experiment,

𝓕, the event space, is a a σ-ring consisting of a set of subsets of Ω that represent events (where an event is defined as a set of outcomes), and

P is a measure function that assigns probabilities to each event in the σ-ring 𝓕 such that P(Ω) = 1.

Since P is a measure function, P(∅) = 0 and P is countably additive:


P({Aₙ | n ∈ ℕ}) = Σ{P(Aₙ) | n ∈ ℕ, Aᵢ ∩ Aⱼ = ∅ ∀ i≠j}.

A typical event space 𝓕 is the power set of the sample space Ω, or the collection of all possible subsets of Ω, including ∅ and Ω itself where the power set of a set A, denoted 2ᴬ is defined as:


2ᴬ = {S ∣ S ⊆ A}.

For a random variable X, the sample space Ω is the set of all possible outcomes of x.


If X is discrete[4]:
Ω = {xₙ | n ∈ ℕ}.

Likewise, if X is continuous and ranges between a and b:
Ω = {x | a ≤ x ≤ b}.

P(x), the probability measure, is the probability that an outcome x will occur; the sum of probabilities of all outcomes is 1:


Σ{P(xₙ) | n ∈ ℕ} = P(Ω) = 1
if X is discrete. P is a countably additive measure function so:
Σ(P({xₙ | n ∈ ℕ})) = P(Ω) = 1.

Likewise, if X is continuous:
∫ P(x) dx = 1
where the integral ranges over the entire domain of X.

To demonstrate, a trivial example of a probability space is used to quantify probabilities associated with possible outcomes of flipping a fair coin. After this random experiment, the coin can either land with the 'heads' side up or alternatively with the opposite 'tails' side facing up. The sample space Ω is simply:


Ω = {heads, tails}.

An event space 𝓕 is constructed from the power set of Ω; 𝓕 = {S ∣ S ⊆ Ω}:


𝓕 = {∅, {heads}, {tails}, {heads, tails}}.

Note that even in this simple example, 𝓕 satisfies all properties required of a σ-ring.


Now, the measure function P assigns a probability to each element of 𝓕 such that P(Ω) = 1. Since the coin is fair, there is equal probability of each outcome:


P({heads}) = P({tails}) = 0.5.

P is necessarily countably additive so:


P({heads}) + P({tails}) = P(Ω) = 1.

Another trivial example quantifies probabilities values associated with the possible outcomes of the random experiment of rolling a fair six-sided die:


Ω = {1, 2, 3, 4, 5, 6},
𝓕 = {S ∣ S ⊆ Ω}, and
P({1}) = P({2}) = P({3}) = P({4}) = P({5}) = P({6}) = 1/6.

There are 2⁶ = 64 subsets in this event space:


𝓕 = {∅,
{1}, {2}, {3}, {4}, {5}, {6},
{1, 2}, {1, 3}, ... , {4, 6}, {5, 6},
{1, 2, 3}, {1, 2, 4}, ... , {3, 5, 6}, {4, 5, 6},
{1, 2, 3, 4}, {1, 2, 3, 5}, ... , {2, 4, 5, 6}, {3, 4, 5, 6},
{1, 2, 3, 4, 5}, {1, 2, 3, 4, 6}, ... , {2, 3, 4, 5, 6}, Ω}.


[4] Definitions of 'discrete' used in different mathematical contexts may vary slightly; here it refers to a random variable that can take on a countable set of distinct values. For a rigorous definition of a discrete set in a general topological space, see: Discrete Set. Eric W Weisstein. MathWorld--A Wolfram Web Resource.
