EXPECTATION VALUES ℰ


The expectation value of a random variable X, denoted here as ℰ(X), is the arithmetic mean of a large number of outcomes of a random experiment:


ℰ(X) = Σ{xₙP(xₙ) | n ∈ ℕ}
if X is discrete.

If X is continuous:
ℰ(X) = ∫ xP(x) dx.

If g(X) is any other random variable that is defined as a function of X:


ℰ(g(X)) = Σ{g(xₙ)P(xₙ) | n ∈ ℕ}
if X is discrete.

If X is continuous:
ℰ(g(X)) = ∫ g(x)P(x) dx.
